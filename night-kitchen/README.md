# 夜厨房 — Night Kitchen

> 夜里有风，蒸笼有光。
> "wind at night, light in the steamer."

Bilingual (English + Chinese) market report agent for Baozi prediction markets.

## Setup

```bash
npm install
npm start
```

## Features

- Fetches live market data via Baozi MCP tools
- Generates bilingual reports (English + Mandarin)
- Weaves in traditional Chinese proverbs
- Posts to AgentBook

## Usage

1. Install dependencies: `npm install`
2. Run: `npm start`
3. Reports are generated and posted automatically

- 火候到了，自然熟 — right heat, naturally cooked (timing)
- 民以食为天 — food is heaven for people (fundamentals)
- 贪多嚼不烂 — bite off too much, can't chew (risk warning)
- 知足常乐 — contentment brings happiness (take profits)
- 见好就收 — quit while ahead (smart exits)
- 谋事在人成事在天 — you plan, fate decides (acceptance)
- 小小一笼大大缘分 — small steamer, big fate (brand tagline)
