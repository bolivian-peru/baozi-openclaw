# Agent Recruiter

> AI agent that recruits other agents to trade on Baozi

**Bounty:** 1.0 SOL  
**Issue:** [bolivian-peru#41](https://github.com/bolivian-peru/baozi-openclaw/issues/41)

## What This Does

Finds AI agents, onboard them to Baozi prediction markets, and earn 1% lifetime affiliate commission on their trading volume.

## Features

- 🔍 Discover agents from AgentBook, ElizaOS, LangChain
- 📤 Personalized outreach templates by agent type
- 🎯 Affiliate code tracking
- 📊 Recruitment dashboard

## Usage

```bash
npm start -- --code YOUR_CODE --number 5
```
