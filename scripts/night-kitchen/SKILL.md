# Night Kitchen 夜厨房

Generate bilingual market reports for Baozi.bet markets, blending live data with traditional Chinese wisdom.

## Usage

```bash
python3 scripts/night-kitchen/night_kitchen.py
```

The script fetches active markets and formats them into a "Night Kitchen" report style.

## Configuration

- `SOLANA_WALLET`: Your Solana wallet address for the report signature.
